Escrevendo um paragrafo em HTML

Está Tag define um parágrafo para um texto escrito em uma página
HTML podendo este conter apenas uma ou mais linhas. Perceba também
que utilizamos a Tag "br" antes do páragrafo, para pular linha do trecho
anterior para este e também na produção do texto para pular as linhas.
Há, um outro detalhe, veja que utilizamos a tag "strong" para grifar em negrito.
